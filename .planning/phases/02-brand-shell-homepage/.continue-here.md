## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Transparent nav on dark heroes | Setting nav initial class to `bg-transparent` makes charcoal links invisible over charcoal/dark hero backgrounds | blocking | Always set `bg-beige/92 backdrop-blur-md` as base class on nav; only adjust on scroll |
| CTA button same color as background | btn-primary is charcoal; placing it on a charcoal section background makes it invisible | blocking | Never override `.newsletter` or CTA section to `var(--charcoal)` — use sage (default) or rose |
| Store registration only in alpine:init | defer-loaded Alpine fires `alpine:init` before ES module runs | blocking | Call `Alpine.store()` directly at module load AND inside `alpine:init` |
